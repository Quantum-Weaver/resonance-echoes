## What It Is

Echoes is your space to log moments with feeling. Music you heard. Movies you watched. Thoughts that passed through. Symptoms you tracked. Dreams you remember. Gratitude you felt.

All stored on your device. No accounts. No cloud. No extraction.

---

## Features

- **Log anything** — Name it, tag it with an emoji, choose a sense category
- **Sensory Lexicon** — Every emoji carries color, sound, texture, and temperature
- **Gentle Insights** — See patterns in your feelings without overwhelming dashboards
- **Sovereign Data** — Export everything as JSON. Purge with one tap. Your data, always.

## Installation

### Prerequisites

- Node.js 18+
- Rust (latest stable)
- Android SDK (for mobile builds)
- Tauri CLI (`npm install -g @tauri-apps/cli`)

### Build

```bash
npm install
npm run tauri android build
```

### Development

```bash
npm run tauri android dev
```
